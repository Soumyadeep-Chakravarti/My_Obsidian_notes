### **Q1 a: Empathy Mapping for a Cordless Mouse** _(12 Marks)_

**Empathy Mapping Definition:**  
Empathy mapping is a tool used to gain deeper insight into a user's feelings, needs, and behaviors. It helps designers and developers create user-centered solutions.

**Empathy Map for Purchasing a Cordless Mouse:**  
The process focuses on what the user _thinks_, _feels_, _says_, and _does_ while interacting with or considering the product.

---

#### **Steps to Create the Empathy Map:**

1. **User Persona:**  
    Define the typical user of the cordless mouse, such as students, office professionals, or gamers.
    
2. **Empathy Map Segments:**
    
    - **Says:**
        
        - "I need a lightweight mouse."
        - "It should have long battery life."
        - "I want smooth connectivity."
    - **Thinks:**
        
        - "Is it comfortable for extended use?"
        - "Will it work well with my laptop/PC?"
        - "Does it justify the price?"
    - **Feels:**
        
        - Excited about wireless freedom.
        - Concerned about lag or connectivity issues.
        - Frustrated with frequent charging or poor ergonomics.
    - **Does:**
        
        - Researches reviews online.
        - Visits stores for hands-on experience.
        - Compares prices and brands.

---

#### **Considerations for the Design of the Mouse:**

1. **Ergonomics:**
    
    - Comfortable grip for long hours.
2. **Battery Life:**
    
    - Rechargeable or replaceable batteries with extended runtime.
3. **Portability:**
    
    - Lightweight and compact design for ease of travel.
4. **Durability:**
    
    - Robust build quality to last for years.
5. **Connectivity:**
    
    - Smooth Bluetooth or USB receiver options.
6. **Aesthetics:**
    
    - Sleek and modern appearance to match user preferences.

### **Q1b: Innovation Flowchart for the Pen Stand Using Human-Centered Design Approach** _(12 Marks)_

**Human-Centered Design Approach**  
This approach focuses on understanding users' needs, challenges, and preferences to create products that are practical, accessible, and user-friendly.

---

#### **Steps to Create the Innovation Flowchart for the Pen Stand**

1. **Empathize:**
    
    - Observe and interview users to understand their needs.
    - Example: Users may want a pen stand that is compact, durable, and visually appealing.
2. **Define:**
    
    - Frame the problem statement based on user insights.
    - Example: “How might we design a pen stand that organizes pens efficiently while being eco-friendly?”
3. **Ideate:**
    
    - Brainstorm multiple design solutions.
    - Example: Modular designs, recycled materials, or multi-functional pen stands.
4. **Prototype:**
    
    - Create low-fidelity prototypes using 3D printing, cardboard, or recyclable materials.
5. **Test:**
    
    - Present prototypes to users for feedback.
    - Refine the design based on feedback.
6. **Implement:**
    
    - Finalize and mass-produce the design.
    - Use sustainable practices and materials.

---

#### **Flowchart Example:**

**Steps Visualization in Flowchart Form:**

- Start  
    → Empathize  
    → Define Problem  
    → Ideate Solutions  
    → Prototype  
    → Test and Collect Feedback  
    → Implement  
    → End
