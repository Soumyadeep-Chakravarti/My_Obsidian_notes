### Que 1: Write a Python program to perform the following tasks:
- Create a list of tuples where each tuple contains a student's name and their corresponding grade.
- Convert this list of tuples into a dictionary where the student's name is the key and the grade is the value.
- Create a set of student names who have grades above 75.
- Print the dictionary and the set.

### Que 2: Write a Python program to validate usernames input by users using regular expressions. Validations are:
- The username must start with a letter.
- The username must contain only letters, numbers, and underscores `(_)`.
- The username must not end with an underscore.
- The username should be between 5 and 12 characters long.
